# vizmat-particles

Open-source particle datasets for `vizmat`.

Used by the visualizer for local testing and web (WASM/GitHub Pages) demos.

- Main project: https://github.com/rs4rse/vizmat
- Live demo (GitHub Pages): https://rs4rse.github.io/vizmat/

## Contents

- `proteins/` – protein structures (`.pdb`, `.xyz`)
- `compounds/` – small-molecule examples (`.sdf`, `.xyz`)
